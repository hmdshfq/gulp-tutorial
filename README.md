# Gulp Tutorial

## Setup

1. Install `node.js`
2. Install PNPM

    ```bash
    npm i -g pnpm
    ```

3. Run the following commands

    ```bash
    pnpm i -g gulp-cli
    pnpm i gulp gulp-concat gulp-sass sass del browser-sync bootstrap 
    ```

4. To run the project type `gulp` into terminal. It will start the project on the `localhost:3000`
